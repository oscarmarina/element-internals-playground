# Element Internals Playground

[ElementInternals API](https://developer.mozilla.org/docs/Web/API/ElementInternals) through reusable mixins.



- **blk-input** & **blk-inline-input**: Simple custom input elements showcasing internal messaging and validity control.
- **BlkMixinElementInternals**: Core mixin to attach `ElementInternals` to any custom element.
- **BlkMixinFormAssociated**: Extension of the core mixin enabling form-associated custom elements.
- **BlkFormValidationEvent**: Extension of the core mixin enabling form-associated custom elements.
